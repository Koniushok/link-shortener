# link-shortener Client
## Installing

```
git clone https://github.com/Koniushok/link-shortener.git
cd link-shortener
yarn
```
## Start dev server
```
yarn start
```
## Note
1. Dev Server runs on port 3000 (can change in  ```src/webpack/development.js[6,15]```)
2. Server URL can change in  ```src\constants\api.js```
